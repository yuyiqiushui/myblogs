# Linux -- hash

hash 命令 —— 显示与清除命令运行时查询的哈希表

> 语法格式： hash 【 参数 】 【 目录 】

常用参数：

| -d       | 在哈希表中清除记录                 |
| -------- | ---------------------------------- |
| -l       | 显示哈希表中的命令                 |
| -p<指令> | 将具有完整路径的命令加入到哈希表中 |
| -r       | 清除哈希表中大的记录               |
| -t       | 显示哈希表中命令的完整路径         |



参考实例：

```
# 显示哈希表中的命令：
hash  -l

# 删除哈希表中的命令：
hash  -r

# 向哈希表中添加命令
hash  -p  /usr/sbin/adduser  myadduser

# 在哈希表中清除记录：
hash  -d

# 显示哈希表中命令的完整路径：
hash  -t
```
