Linux - rm

## rm 命令 - 移出文件或目录

> 语法格式： rm 【 参数 】 【 文件 】

常用参数：

| -f   | 忽略不存在的文件，不会出现警告信息 |
| ---- | ---------------------------------- |
| -i   | 删除前会询问用户是否操作           |
| -r/R | 递归删除                           |
| -v   | 显示指令的详细执行过程             |



参考实例：

```
# 删除前逐一询问确认：
rm  -i  test.txt.bz2
rm: remove  regular  file  `test.txt.bz2`?

# 直接删除，不会有任何提示：
rm  -f  test.txt.baz

# 递归删除目录及目录下所有文件
mkdir  /data/log
rm  -rf  /data/log

# 删除当前目录下所有文件：
rm  -rf  *

清空系统中所有文件 ( 谨慎 )：
rm  -rf  /*
```
