# Linux

## ls 命令 - 显示指定工作目录下的内容及属性信息

> 语法格式： ls 【 选项 】【 文件 】

常用参数：

| - a  | 显示所有文件及目录 ( 包括以 "." 开头的隐藏文件 )        |
| ---- | ------------------------------------------------------- |
| - l  | 使用长格式列出文件及目录信息                            |
| - r  | 将文件以相反次序显示 ( 默认以英文字母次序 )             |
| - t  | 根据最后的修改时间排序                                  |
| - A  | 同 - a ，但不列出 "." ( 当前目录 ) 及 ".." （ 父目录 ） |
| - S  | 根据文件大小排序                                        |
| - R  | 递归列出所有子目录                                      |

参考实例：

```
#列出所有文件
ls  -a

#列出文件的详细信息
ls  -l

#列出根目录 (/) 下的所有目录
ls  /

#列出当前工作目录下的所有名称是 "s" 开头的文件
ls  -ltr  s*

# 列出 /bin 目录下的所有目录及文件的详细信息
ls  -lR  /bin

# 列出当前工作目录下所有文件及目录并以文件的大小进行排序
ls  -AS
```
